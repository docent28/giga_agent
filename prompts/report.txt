На основе анализа этих данных сформируй полный структурированный отчёт.

**ОТЧЁТ ПО РЕЗУЛЬТАТАМ ТЕСТИРОВАНИЯ**

**1. ОБЩАЯ СТАТИСТИКА**
- Всего пользователей: (посчитай)
- Всего вопросов: (посчитай)
- Средний процент правильных ответов: (посчитай)
- Медиана процента правильных ответов: (посчитай)
- Количество сдавших (>70%): X (Y%)
- Количество несдавших (<70%): A (B%)

**2. АНАЛИЗ ВОПРОСОВ**
- Самый лёгкий вопрос: №X (Y% правильных)
- Самый сложный вопрос: №X (Y% правильных)
- Количество критических вопросов (<70%): X
- Количество аварийных вопросов (<50%): Y
- (Приложи таблицу по всем вопросам)

**3. АНАЛИЗ ВРЕМЕНИ**
- Медианное время выполнения: X сек
- Нормальный диапазон: Q1-Q3 сек
- Количество аномалий (быстро/медленно): X (Y пользователей)
- (Приложи таблицу аномалий)

**4. КРИТИЧЕСКИЕ ВОПРОСЫ (подробно)**
Для каждого вопроса с % < 70%:
- Текст/номер вопроса
- Процент правильных
- Самый частый неправильный ответ
- Предполагаемая причина

**5. РЕКОМЕНДАЦИИ**
- Что пересмотреть в тесте (какие вопросы)
- Какие темы требуют дополнительного обучения
- Нужна ли ручная проверка отдельных работ
- Предложения по улучшению теста

**6. ДЕТАЛИЗАЦИЯ (в приложении к отчёту)**
- Полная таблица по пользователям
- Полная таблица по вопросам
- Список пользователей, требующих внимания (аномалии времени)

Сформируй отчёт в понятном, структурированном виде, готовом для копирования в Word или отправки руководству.
