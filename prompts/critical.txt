**Базовая статистика по данным:**
{basic_stats}

Выполни следующие действия:

1. **Анализ успешности по вопросам:**
   - Процент правильных ответов на каждый вопрос
   - Вопросы с % < 70% — отметить как «критические»
   - Вопросы с % < 50% — отметить как «аварийные»

2. **Анализ паттернов ошибок:**
   - Если в вопросе несколько вариантов ответа — определи, какой неправильный ответ выбирают чаще всего
   - Есть ли вопросы, которые пользователи массово пропускают?
   - Есть ли закономерности в ошибках (например, все ошибаются в одной теме)?

3. **Анализ изменений во времени (если есть несколько периодов):**
   - Сравни текущие результаты с предыдущими (если в данных есть даты)
   - Выдели вопросы, где процент правильных резко упал

4. **Рекомендации:**
   - Какие вопросы требуют переформулировки?
   - Какие темы нужно объяснить заново?
   - Нужно ли добавить подсказки или изменить сложность?

Выдай результат в виде:
